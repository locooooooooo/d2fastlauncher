# 暗黑破坏神2：重制版 快捷多开器 (D2R Fast Launcher)

一款基于 **Tauri + Vue3 + Rust** 打造的现代化、跨平台暗黑2重制版快捷登录与多开工具。专为国服（及其他区服）玩家设计，解决战网繁琐登录流程、轻松实现游戏多开！

## ✨ 核心特性

- 🚀 **现代轻量化 UI**：基于 Vue3 + Tailwind CSS，界面清爽，操作直观。
- 🛡️ **底层硬核多开**：通过 Rust 底层调用 Windows Native API (NtQuerySystemInformation / DuplicateHandle) 精准关闭战网及游戏的防多开 Mutex 锁，**无需复制多个游戏客户端**即可无限多开！
- 🌍 **全区服支持**：
  - **国际服（自动模式）**：绕过战网，直接拉起 `D2R.exe`，实现极速秒进。
  - **国服特化（战网模式）**：针对国服需要手机验证码和浏览器登录的特殊要求，一键拉起并修改战网配置，支持战网多开，手动通过战网进入游戏。
- 📂 **智能路径识别**：通过读取注册表，自动检测您的战网客户端与游戏安装路径，开箱即用。
- 💾 **账号安全管理**：本地持久化存储账号与密码，自由增删改查。

## 🛠️ 技术栈

- **前端**：Vue 3 + TypeScript + Pinia + Tailwind CSS
- **后端**：Rust + Tauri
- **系统 API**：Windows API (`ntapi`, `winapi`, `winreg`)

## 📦 如何使用

1. **下载程序**：在 Releases 页面下载最新版本的可执行文件（或自行克隆编译）。
2. **添加账号**：打开工具，点击“添加账号”，选择您的游戏区服（国服/美服/亚服/欧服等）。
   - *注意：国服账号请选择【战网启动】，国际服账号可选择【自动启动】。*
3. **启动游戏**：点击账号卡片上的“启动游戏”按钮。
4. **一键解除多开限制**：在启动下一个账号前，或者遇到“游戏已在运行”的提示时，点击顶部的【一键解除多开限制】按钮，即可继续开启下一个游戏窗口！

## ⚠️ 注意事项

- 本工具通过合法解除系统互斥锁（Mutex）实现多开，**不修改任何游戏内存和核心文件**，理论上无封号风险。但作为第三方辅助工具，**一切潜在风险请用户自行承担**。
- 【一键解除多开限制】功能需要管理员权限来关闭系统底层的 Handle 句柄，若遇到杀毒软件误报，请添加白名单。
- 使用【自动启动】跳过战网直接进游戏时，请确保战网客户端处于完全关闭状态，否则可能会产生冲突。
